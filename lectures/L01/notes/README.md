# L01 - Anteckningar

Kretsscheman från dagens lektion:
* [gates.png](./gates.png) demonstrerar de med grundläggande logiska grindarna AND, OR och NOT
med tryckknappar och lysdioder. Schemat kan simuleras i [CircuitVerse](https://circuitverse.org/simulator) genom att importera filen [gates.cv](./gates.cv). 
* [adas.png](./adas.png) demonstrerar ett ADAS-system (*Advanced Driver Assistance System*) för ett
fordon. Schemat kan simuleras i [CircuitVerse](https://circuitverse.org/simulator) genom att importera filen [adas.cv](./adas.cv).

## Simulering i CircuitVerse
Respektive kretsschema kan simuleras i [CircuitVerse](https://circuitverse.org/simulator) genom att importera motsvarande .cv-fil, exempelvis [adas.cv](./adas.cv). För att göra detta, öppna CircuitVerse, klicka på `Project->Import file` och välj filen i fråga.
